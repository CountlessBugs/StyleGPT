# 穿搭GPT - 快速使用指南

## 🚀 立即开始使用

您的穿搭GPT应用已经成功启动！现在可以通过预览浏览器访问应用了。

### 第一步：配置 API 密钥

在使用AI功能之前，需要配置 OpenAI API 密钥：

1. 复制 `.env.example` 文件为 `.env`
2. 在 `.env` 文件中填入您的 OpenAI API 密钥：

**选项 A：使用官方 OpenAI API**
```env
OPENAI_API_KEY=sk-your-actual-api-key-here
# OPENAI_API_BASE_URL 留空或不填
```

**选项 B：使用第三方兼容 API（推荐，更便宜且国内访问更快）**

使用 DeepSeek（推荐）：
```env
OPENAI_API_KEY=your_deepseek_api_key
OPENAI_API_BASE_URL=https://api.deepseek.com/v1
```

使用智谱 AI：
```env
OPENAI_API_KEY=your_zhipu_api_key
OPENAI_API_BASE_URL=https://open.bigmodel.cn/api/paas/v4
```

使用硅基流动：
```env
OPENAI_API_KEY=your_siliconflow_api_key
OPENAI_API_BASE_URL=https://api.siliconflow.cn/v1
```

3. 保存文件后，重启开发服务器（Ctrl+C 然后再次运行 `npm run dev`）

> 💡 **提示**：第三方 API 通常价格更低（如 DeepSeek 约为官方 OpenAI 的 1/10），且在国内访问速度更快、更稳定。

### 功能演示

#### 0️⃣ 我的衣柜

**使用场景**：一次性添加所有衣物，方便后续使用

**步骤**：
1. 点击"添加衣物"按钮
2. 填写衣物信息（名称、分类、颜色、描述）
3. 保存后即可在衣柜中看到
4. 使用类别筛选和搜索功能管理衣物
5. 点击"导入到穿搭方案"一键导入

**示例数据**：
- 名称：白色T恤
- 分类：上衣
- 颜色：白色
- 描述：纯棉材质，宽松版型

**功能亮点**：
- ✅ 数据自动保存在浏览器，无需登录
- ✅ 支持7种分类，方便管理
- ✅ 实时搜索和筛选
- ✅ 一键导入到穿搭方案

#### 1️⃣ 穿搭方案生成

**使用场景**：你已经有一些衣服，但不知道怎么搭配

**示例输入**：
- **穿搭风格**：简约通勤
- **已有服装**：白色衬衫、黑色西装裤、米色毛衣、蓝色牛仔裤、白色运动鞋（或从衣柜导入）
- **穿搭场合**：上班

**AI 会提供**：
- 3-5套完整的搭配方案
- 每套搭配的详细说明
- 适合的场合和季节
- 配饰建议

#### 2️⃣ 购买建议

**使用场景**：想买新衣服，需要专业建议

**示例输入**：
- **穿搭风格**：韩系
- **计划购买**：想买一件春季外套和一条百搭的裤子
- **预算**：1000-2000元

**AI 会提供**：
- 具体单品推荐（款式、颜色、材质）
- 购买优先级排序
- 搭配建议和实用性分析
- 预算分配建议
- 品牌和购买渠道建议

#### 3️⃣ 虚拟试穿

**使用场景**：想看看穿某套衣服的效果

**示例输入**：
- **上传照片**：一张正面全身照
- **服装描述**：黑色西装外套配白色衬衫和深蓝色牛仔裤

**AI 会生成**：
- 虚拟试穿效果图

> 注意：虚拟试穿需要配置 Nano Banana API 密钥。如果暂时没有，可以先使用前两个功能。

### 💡 获得更好建议的技巧

1. **详细描述**：越详细的描述，AI 能提供越精准的建议
2. **颜色和材质**：说明衣服的颜色、材质（如：纯棉、毛呢等）
3. **具体场合**：明确使用场景（如：日常上班、周末休闲、正式场合等）
4. **个人偏好**：可以说明你的喜好（如：喜欢宽松的、避免太花哨的等）

### 📱 界面导航

- 顶部有四个标签页：**我的衣柜**、**穿搭方案**、**购买建议**、**虚拟试穿**
- 点击不同标签切换功能
- 每个功能都有清晰的表单指引
- 填写信息后点击按钮即可获得AI建议

### ⚠️ 重要提示

1. **首次使用**必须配置 OpenAI API 密钥才能使用AI功能
2. **API 费用**：使用 OpenAI API 会产生费用，每次调用约 $0.002-0.01
3. **响应时间**：通常 5-15 秒可获得结果
4. **虚拟试穿**：需要额外配置 Nano Banana API（可选功能）

### 🔧 故障排除

**问题：点击生成按钮没有反应**
- 检查是否已配置 API 密钥
- 打开浏览器开发者工具（F12）查看错误信息
- 确认网络连接正常

**问题：显示 API 错误**
- 验证 API 密钥是否正确
- 检查 OpenAI 账户是否有余额
- 确认 API 密钥权限正常

**问题：生成的建议不够好**
- 尝试提供更详细的描述
- 使用更具体的风格关键词
- 多次生成选择最佳结果
