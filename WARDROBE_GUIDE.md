# 衣柜功能使用指南

## 📖 功能概述

"我的衣柜"功能允许你持久化保存和管理所有的衣物，让你可以：
- 📝 记录所有拥有的衣物
- 🔍 快速搜索和筛选
- 📋 一键导入到穿搭方案
- 💾 数据自动保存在本地

## 🚀 快速开始

### 1. 添加第一件衣物

1. 点击主界面的 **"👔 我的衣柜"** 标签
2. 点击右上角的 **"➕ 添加衣物"** 按钮
3. 在弹出的对话框中填写信息：

**必填字段：**
- **衣物名称**：简短描述，如"白色T恤"、"蓝色牛仔裤"

**可选字段：**
- **分类**：选择适合的类别（上衣/裤装/裙装/外套/鞋履/配饰/其他）
- **颜色**：衣物的主要颜色
- **描述**：更详细的信息，如材质、版型、品牌等

4. 点击 **"添加"** 按钮保存

### 2. 管理衣物

#### 查看衣物列表
- 所有衣物以卡片形式展示
- 每个卡片显示：名称、分类、颜色、描述、添加日期
- 统计信息显示在顶部

#### 筛选衣物
点击分类按钮快速筛选：
- **全部**：显示所有衣物
- **上衣**：T恤、衬衫、毛衣等
- **裤装**：牛仔裤、西装裤、休闲裤等
- **裙装**：各种裙子
- **外套**：夹克、大衣、风衣等
- **鞋履**：运动鞋、皮鞋、靴子等
- **配饰**：包包、帽子、围巾等
- **其他**：未分类的物品

每个分类按钮会显示该分类下的衣物数量。

#### 搜索衣物
在搜索框中输入关键词，可以搜索：
- 衣物名称
- 颜色
- 描述内容

搜索是实时的，无需点击按钮。

#### 删除衣物
点击衣物卡片右上角的 **🗑️** 图标，确认后即可删除。

> ⚠️ **注意**：删除操作不可恢复，请谨慎操作！

### 3. 导入到穿搭方案

这是衣柜功能最实用的特性！

**操作步骤：**
1. （可选）先筛选或搜索特定衣物
2. 点击 **"📋 导入到穿搭方案"** 按钮
3. 系统会自动将当前显示的所有衣物导入
4. 切换到 **"穿搭方案"** 标签
5. 你会看到"已有服装"字段已自动填充

**使用场景：**
- 想要基于所有衣物生成搭配 → 不筛选，直接导入全部
- 只想用特定季节的衣物 → 先筛选再导入
- 为特定场合搭配 → 搜索相关衣物后导入

## 💡 实用技巧

### 1. 合理分类
建议按照实际使用习惯分类：
```
上衣：T恤、衬衫、毛衣、背心
裤装：牛仔裤、西装裤、运动裤、短裤
裙装：连衣裙、半身裙、长裙、短裙
外套：西装外套、风衣、羽绒服、夹克
鞋履：运动鞋、皮鞋、靴子、凉鞋
配饰：包包、帽子、围巾、腰带、首饰
```

### 2. 详细描述
在描述中可以包含：
- **材质**：纯棉、毛呢、真丝、涤纶等
- **版型**：宽松、修身、oversize等
- **风格**：休闲、正式、运动、复古等
- **季节**：春秋、夏季、冬季
- **场合**：通勤、约会、运动、居家
- **品牌**：如有需要

示例：
```
名称：白色衬衫
分类：上衣
颜色：白色
描述：纯棉材质，修身版型，适合正式场合，春夏秋三季可穿，优衣库
```

### 3. 颜色标注
准确标注颜色有助于AI生成更好的搭配建议：
- 基础色：黑、白、灰、米、驼
- 常用色：蓝、红、绿、黄、粉、紫
- 组合色：黑白格纹、蓝白条纹

### 4. 定期整理
建议每季度整理一次：
1. 删除已经不穿的衣物
2. 添加新购买的衣物
3. 更新描述信息

### 5. 搭配使用
最佳工作流程：
```
1. 在"我的衣柜"中维护所有衣物
   ↓
2. 根据需要筛选/搜索
   ↓
3. 导入到"穿搭方案"
   ↓
4. 填写风格和场合
   ↓
5. 生成AI搭配建议
```

## 🔧 数据管理

### 数据存储
- 衣柜数据保存在浏览器的 **localStorage** 中
- 数据仅存储在本地，不会上传到服务器
- 每次修改会自动保存

### 数据安全
⚠️ **重要提示**：

**数据会丢失的情况：**
- 清除浏览器缓存/Cookie
- 使用浏览器的"清除浏览数据"功能
- 卸载浏览器
- 使用无痕模式（数据不会保存）

**保护数据的方法：**
1. 不要随意清除浏览器数据
2. 使用同一个浏览器访问应用
3. 定期截图保存重要信息
4. 将数据导出为文本备份（可以复制导入的文本）

### 数据迁移
如果需要更换设备或浏览器：

**导出方法：**
1. 打开浏览器开发者工具（F12）
2. 进入 Console（控制台）
3. 输入：`console.log(localStorage.getItem('wardrobe'))`
4. 复制输出的内容保存

**导入方法：**
1. 在新设备/浏览器中打开应用
2. 打开浏览器开发者工具（F12）
3. 进入 Console（控制台）
4. 输入：`localStorage.setItem('wardrobe', '粘贴之前复制的内容')`
5. 刷新页面

> 💡 **提示**：未来版本将支持云端同步和一键导出/导入功能

## ❓ 常见问题

### Q1: 数据会丢失吗？
A: 数据保存在浏览器本地，只要不清除浏览器数据就不会丢失。建议定期备份重要数据。

### Q2: 可以添加多少件衣物？
A: 理论上没有限制，但为了性能考虑，建议控制在 500 件以内。

### Q3: 可以上传衣物图片吗？
A: 当前版本暂不支持图片上传，这是未来版本的计划功能。

### Q4: 可以导出衣柜数据吗？
A: 可以通过浏览器开发者工具手动导出，未来版本将提供一键导出功能。

### Q5: 可以和朋友分享我的衣柜吗？
A: 当前版本不支持分享，但你可以截图或复制导出的文本分享。

### Q6: 数据会同步到其他设备吗？
A: 当前版本仅支持本地存储，不支持多设备同步。云端同步功能在开发计划中。

### Q7: 删除的衣物可以恢复吗？
A: 无法恢复，删除前请确认。建议重要衣物删除前先记录。

### Q8: 为什么搜索不到某件衣物？
A: 检查拼写是否正确，或尝试搜索其他关键词（颜色、描述等）。

## 🎯 最佳实践

### 1. 建立完整的衣柜档案
首次使用时，建议：
- 花 30-60 分钟完整录入所有衣物
- 从常穿的开始录入
- 分批次录入，避免疲劳

### 2. 保持更新
- 购买新衣物后立即添加
- 淘汰旧衣物时及时删除
- 换季时检查更新

### 3. 合理利用筛选
- 换季时筛选当季衣物
- 准备特定场合时筛选相关衣物
- 定期查看各分类数量，了解衣柜结构

### 4. 结合AI建议
- 导入全部衣物让AI给出整体搭配建议
- 导入特定衣物尝试新的搭配组合
- 根据AI建议发现之前忽略的搭配可能
